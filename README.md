# Rocket Game


## About

Rocket Game is a lightweight browser game where the player controls a rocket and must avoid incoming obstacles with arrow keys and collect white dots to up their score and level up . The game is implemented in JavaScript and HTML.
---

## Features
As you maneuver and get further in the game, the game sounds will change as you move onto the next level and the game will also get harder with increased obstacle spawning, and a higher speed.
---

## How to Play

https://madfish89.github.io/rocket_game/
---

## Controls

- Arrow Up / W / Space: Move up / thrust
- Left / Right arrow : move horizontally
- r: restart
